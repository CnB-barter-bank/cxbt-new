# CXBT Frontend MVP

Vue3+Quasar+Pinia приложение с поддержкой мобильных устройств и полным покрытием тестами.

## 📋 Обзор проекта

CXBT Frontend - это современное веб-приложение, созданное с использованием передовых технологий для обеспечения высокой производительности, удобства разработки и отличного пользовательского опыта.

### 🎯 Ключевые характеристики

- **100% покрытие тестами** - все компоненты, маршрутизатор и store полностью протестированы
- **TypeScript** - строгая типизация для надежности кода
- **Quasar Framework** - кроссплатформенность и готовые компоненты
- **Pinia** - современное управление состоянием
- **Vitest** - быстрое и надежное тестирование

## 🛠 Технологический стек

### Основные зависимости
- **Vue 3.4.0** - прогрессивный JavaScript-фреймворк
- **Quasar 2.14.0** - Vue-фреймворк для создания кроссплатформенных приложений
- **Pinia 2.1.7** - официальное решение для управления состоянием в Vue 3
- **Vue Router 4.2.5** - официальная библиотека маршрутизации для Vue.js

### Инструменты разработки
- **TypeScript 5.3.3** - статическая типизация JavaScript
- **Vite 2.9.18** - быстрый сборщик модулей
- **Vitest 1.6.0** - тестовый фреймворк, совместимый с Vite
- **ESLint 8.57.0** - инструмент для анализа кода
- **Prettier 3.2.5** - форматировщик кода

### Тестирование
- **@vue/test-utils 2.4.5** - официальные утилиты для тестирования Vue компонентов
- **happy-dom 14.7.1** - легковесная реализация DOM для тестирования
- **@vitest/coverage-v8 1.6.0** - покрытие кода тестами

## 📦 Установка

```bash
npm install
```

## 🚀 Запуск разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:9000`

## 🧪 Тестирование

### Запуск всех тестов
```bash
npm run test
```

### Запуск тестов с UI
```bash
npm run test:ui
```

### Запуск тестов с покрытием
```bash
npm run test:coverage
```
